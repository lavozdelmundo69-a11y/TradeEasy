# TradeEasy - App de Aprendizaje de Trading

## Prompt Original del Proyecto

Actor: Experto en trading, diseño UX, React Native/Expo

Objetivo: Crear app móvil tipo Duolingo para aprender trading desde cero.

---

## Contenido del Prompt

Te paso un prompt de un nuevo proyecto en un archivo. Crea una nueva carpeta para contener todo de el que se llama Actúa como un experto en: * trading y análisis técnico * diseño de producto educativo * UX para apps móviles * arquitectura de aplicaciones React Native con Expo Estoy creando **un proyecto de app móvil para aprender trading desde cero**, similar a Duolingo pero enfocado en trading. La app aún **no está desarrollada**, por lo que necesito ayuda para diseñar **toda la arquitectura del producto, el sistema educativo y el contenido**. Tu objetivo es ayudarme a diseñar **la app completa**, incluyendo: 1. estructura de aprendizaje 2. sistema educativo 3. contenido de lecciones 4. ejercicios interactivos 5. simulaciones de mercado 6. sistema de progresión 7. arquitectura de la app 8. estructura de datos para React Native La app será desarrollada usando **React Native con Expo**. El contenido debe estar optimizado para **pantallas móviles y sesiones cortas (2-5 minutos)**. --- OBJETIVO DE LA APP La app debe permitir a los usuarios: * aprender trading desde cero * entender cómo funcionan los mercados * practicar análisis de mercado * tomar decisiones simuladas de trading * responder quizzes interactivos * avanzar por niveles * desbloquear nuevos conceptos El aprendizaje debe ser **progresivo, interactivo y gamificado**. --- SISTEMA DE APRENDIZAJE Divide el contenido en **3 niveles principales**: NIVEL 1 – FUNDAMENTOS NIVEL 2 – COMPRENSIÓN DEL MERCADO NIVEL 3 – ENFOQUES AVANZADOS Cada nivel debe contener entre **15 y 30 lecciones cortas**. --- FORMATO DE CADA LECCIÓN Cada lección debe incluir: * id * nivel * título * explicación simple (máximo 3-5 párrafos cortos) * ejemplo práctico * ejercicio interactivo * opciones de respuesta * respuesta correcta * explicación de la respuesta * resumen corto El contenido debe ser **muy fácil de leer en móvil**. --- TIPOS DE EJERCICIOS INTERACTIVOS Incluye diferentes tipos de ejercicios. QUIZ DE CONCEPTOS Ejemplo: Situación: "El precio rebota varias veces en la misma zona." Pregunta: ¿Qué representa esta zona? Opciones: * Soporte * Resistencia * Tendencia --- IDENTIFICACIÓN DE TENDENCIA Escenario: HL → HH → HL → HH Pregunta: ¿Qué tendencia hay? Opciones: * Alcista * Bajista * Lateral --- DECISIÓN DE TRADING Situación: "El precio llega a una resistencia fuerte después de una subida rápida." Pregunta: ¿Qué decisión sería más prudente? Opciones: * Comprar * Vender * Esperar confirmación --- INTERPRETACIÓN DE VELAS Situación: "Vela con mecha superior larga y cuerpo pequeño." Pregunta: ¿Qué puede indicar? --- SIMULACIÓN DE MERCADO Describe escenarios como: * tendencia alcista * soporte * ruptura * consolidación * toma de liquidez Y pregunta qué haría el usuario: * comprar * vender * esperar --- NIVEL 1 – FUNDAMENTOS Explica conceptos básicos: * qué es trading * mercados financieros * precio * oferta y demanda * velas japonesas * partes de una vela * velas alcistas y bajistas * patrones básicos de velas * soportes y resistencias * tendencias * volumen * liquidez básica * timeframes * gestión del riesgo * stop loss * risk reward Los ejercicios deben ser **muy simples**. --- NIVEL 2 – COMPRENSIÓN DEL MERCADO Explica: * estructura del mercado * HH HL LH LL * Break of Structure * Change of Character * consolidaciones * falsas rupturas * zonas de oferta y demanda * liquidez * confluencias * contexto de mercado * psicología del trading Los ejercicios deben usar **escenarios más realistas**. --- NIVEL 3 – CONCEPTOS AVANZADOS Explica enfoques profesionales como: * Wyckoff * ICT * Smart Money Concepts * SMT divergences * Order Blocks * Liquidity Pools * Fair Value Gaps * manipulación institucional Para cada método explica: * filosofía * conceptos principales * cómo se relaciona con soportes, estructura y liquidez Incluye **simulaciones de mercado más complejas**. --- SISTEMA DE GAMIFICACIÓN Diseña un sistema similar a Duolingo: * XP por lección * racha diaria * niveles desbloqueables * progreso visual * recompensas por completar módulos --- DIFICULTAD ADAPTATIVA Define reglas como: Si el usuario falla muchas preguntas: * mostrar ejercicios más simples Si responde correctamente varias seguidas: * mostrar escenarios más complejos --- SIMULACIONES DE GRÁFICOS Cuando describas gráficos usa representaciones simples que luego puedan convertirse en gráficos visuales. Ejemplo: "Tendencia alcista → retroceso al soporte → vela alcista fuerte" --- ARQUITECTURA DE LA APP (React Native + Expo) Propón la estructura del proyecto: pantallas principales: * Home * Mapa de aprendizaje (tipo Duolingo) * Lección * Ejercicio interactivo * Resultado * Perfil de usuario * Progreso * Estadísticas --- COMPONENTES PRINCIPALES Sugiere componentes reutilizables como: * LessonCard * QuizCard * MarketScenario * ProgressBar * XPIndicator * LevelMap * AnswerButtons --- ESTRUCTURA DE DATOS Define cómo deberían estructurarse las lecciones en JSON para poder usarlas en la app. Ejemplo: { id: "lesson_01", level: "basic", title: "Soportes y resistencias", explanation: "...", example: "...", question: "...", options: ["Soporte", "Resistencia", "Tendencia"], correctAnswer: "Soporte", explanationAnswer: "...", summary: "..." } --- MVP DE LA APP Sugiere cómo construir un **MVP simple** que incluya: * 10 lecciones * sistema de progreso * quizzes interactivos * mapa de niveles básico --- OBJETIVO FINAL Diseñar el **sistema educativo completo y la base técnica de una app móvil para aprender trading**, optimizada para **React Native con Expo**, con aprendizaje progresivo, ejercicios interactivos y simulaciones de mercado.
